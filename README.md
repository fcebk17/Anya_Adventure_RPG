## Set up Express Enviroment

```
$npm init
```

# Install EJS

```
$ npm ejs
$ npm install ejs --save
$ npm install ejs-locals --save
```

## Install Express

```
$ npm install express --save

```

## Install nodemon

```
$ npm install nodemon --save-dev
```

## Install Static Server
```
$ npm install -g node-static
$ npm install node-static --save
```
